myFlix is a server-side web application, providing users with access to information about differnt movies, actors, directors and their specific genre. ENabling users to sign up, update their personal information and create list of their favourite movies.


Features

  • Return a list of ALL movies
  • Return data i.e. description, genre, director, image URL about a single movie by title
  • Return data about a genre by name/title e.g. “Comedy”
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info by their username
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister


Table below details all elements of endpoints created in myFlix

Business Logic URL Parameters in URL HTTP Method Request Data Format Response Data Format
Return a list of ALL movies to the user http://localhost:8080/movies - Get - JSON object holding data about all the movies
Return data about a single movie by title http://localhost:8080/movies/:title :title Get - JSON object holding data about a single movie
Return data about a genre by title http://localhost:8080/movies/genre/:genreName :genreName Get - JSON object holding data about a specific genre of moive
Return data about a director including bio, birth year, death year by their name http://localhost:8080/movies/Directors/:directorName :directorName Get - JSON object holding data about a specific director
Allow new users to register http://localhost:8080/users - Post JSON object holding data about user to add, structured like: { "userName" : "string" , "password" : "string" , "email" : "string" , "birthdate" : "date" } JSON object holding data about user that was added, structured like: { "userName" : "Nathan" , "password" : "Backbone" , "email" : "nahan@hotmail.com" , "birthdate" : "1980-06-11" , "favouriteMovies" : [] , "_id" : "6400..."
Allow users to update their user info via username http://localhost:8080/users/:userName :username Put - JSON object holding data about the updated user information
Allow users to add a movie to their list of favorites http://localhost:8080/users/:userName/movies/:movieId :userName, :movieId Post - JSON object holding data about the user and the movie that was added
Allow users to remove a movie from their list of favorites http://localhost:8080/users/:userName/movies/:movieId id / movieTitle:userName, :movieId Delete - JSON object holding data about the user with the movie now removed
Allow existing users to deregister http://localhost:8080/users/:userName :userName Delete - Text describing user being removed
